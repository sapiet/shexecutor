# Shexecutor

Configuration files based executions for Shell.

Require Composer & PHP or NPM & NodeJS


## Installation

for PHP (fastest)

```composer install```

for NodeJS

```npm install```


## Configuration

Copy `environments.yaml.dist` to `environments.yaml` and configure targets


## Commands

```./ssh [target]```

```./syncdatabase [target]```

```./syncfiles [target]```

```./exec [target] [command]```
